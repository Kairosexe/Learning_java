Utiliza la estructura de control SWITCH e IF en el lenguaje Java para crear un carrito de mercado, el carrito preguntará las cantidades, con los siguientes productos 

NOTA: Está permitido usar las siguientes clases en esta actividad:
- BufferedReader
- Scanner

Queso a     $157 unidad
